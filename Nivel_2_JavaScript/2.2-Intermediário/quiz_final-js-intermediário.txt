Sua resposta
Verdadeira

Explicação:

Questão 3
CORRETA

Abaixo temos um exemplo de Operador de coalescência. Qual o retorno?

let content = null
console.log(content ?? "Teste")

Sua resposta
A palavra Teste porque content é nulo;

Explicação:
Parabéns, você acertou! O operador de coalescência nula (??) retorna o operando do lado direito quando o operando do lado esquerdo é null ou undefined. Neste caso, como content é null, o valor "Teste" é retornado.

Questão 7
CORRETA

Sobre os métodos push() e pop() é correto afirmar:

Sua resposta
O método push() adiciona um item ao final do array enquanto o método pop() remove um item do final do array;

Explicação:
Parabéns, você acertou! O método push() realmente é utilizado para adicionar um ou mais elementos ao final de um array, modificando o array original e retornando o novo comprimento do array. Já o método pop() remove o último elemento de um array e retorna esse elemento, também modificando o array original. Esses métodos são fundamentais para a manipulação de estruturas de dados do tipo pilha (stack) em JavaScript, onde a inserção e remoção ocorrem sempre em uma das extremidades.

Qual das alternativas abaixo é uma estrutura correta de um laço for que imprime na tela os números de 0 a 10?

Sua resposta
for (let i = 0; i <= 10; i++) { 
   console.log(i)
}

Explicação:
Parabéns! Você acertou a alternativa. Esta estrutura de laço for está correta para imprimir os números de 0 a 10. Ele inicializa i em 0, continua enquanto i for menor ou igual a 10 (incluindo o 10), e incrementa i a cada iteração, garantindo que todos os números no intervalo de 0 a 10 sejam exibidos no console.
