# 🚀 NÍVEL 1 — Fundamentos do HTML e do CSS

## 📘 Fundamentos do HTML  
**MÓDULO**  
32 AULAS | 5 ATIVIDADES | 1h 28min  

Nesse módulo você aprende os conceitos essenciais do HTML: funcionamento das tags, uso de atributos, estruturação de páginas, aplicação de semântica, links, listas, imagens e caminhos de arquivos. É o primeiro passo para construir páginas bem organizadas e entender como a web funciona.

## 🎨 Fundamentos do CSS  
**MÓDULO**  
33 AULAS | 4 ATIVIDADES | 2h 10min  

Aqui você aprende o que é CSS, como adicionar estilos a páginas web, entender o box model, estilizar textos, trabalhar com cores, backgrounds e organização visual.

## 🍽️ Projeto: Página de Receita  
**MÓDULO**  
10 AULAS | 56min  

Aplicação prática dos fundamentos de HTML e CSS na construção de uma página completa de receita.

---

## 🌍 Desafio Prático: Local Turístico  
**OPCIONAL**  

Desenvolvimento de uma página web com informações sobre um local turístico, aplicando estruturação e estilização.

## 📝 Quiz Avaliativo — Fundamentos do HTML e CSS  
**OBRIGATÓRIO**  
12 QUESTÕES  

Avaliação dos conhecimentos adquiridos no nível 1.

# ⚡ NÍVEL 2 — JavaScript

## 🟡 JavaScript Básico  
**MÓDULO**  
82 AULAS | 6 ATIVIDADES | 7h 20min  

Exploração dos fundamentos do JavaScript: funções, manipulação do DOM, eventos, escopo, expressões regulares e depuração. Inclui a criação de um aplicativo de conversor de moedas.

## 🛒 Desafio Prático: Lista de Compras  
**OPCIONAL**  

Criação de uma aplicação interativa de lista de compras utilizando JavaScript.


## 📝 Quiz Avaliativo — JavaScript Básico  
**OBRIGATÓRIO**  
15 QUESTÕES  

Avaliação dos conteúdos estudados no módulo básico.

## 🟠 JavaScript Intermediário  
**MÓDULO**  
78 AULAS | 6 ATIVIDADES | 6h 11min  

Estudo de métodos de manipulação de texto, estruturas de repetição eficientes, manipulação de datas e horas, além da introdução a classes para organização e modularização do código.


## 🎲 Desafio Prático: Sorteador de Números  
**OPCIONAL**  

Desenvolvimento de um sorteador de números aplicando conceitos intermediários.


## 📝 Quiz Avaliativo — JavaScript Intermediário  
**OBRIGATÓRIO**  
15 QUESTÕES  

Avaliação dos conhecimentos intermediários.

## 🔥 JavaScript Antes do Framework  
**MÓDULO**  
69 AULAS | 5h 48min  

Estudo de JavaScript moderno: imutabilidade, módulos, funções assíncronas, pacotes, APIs, compiladores e bundlers. Base essencial para compreender frameworks como React.

# 🔵 NÍVEL 3 — TypeScript

## 📘 Conhecendo o TypeScript  
**MÓDULO**
